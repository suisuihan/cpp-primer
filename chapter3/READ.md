##3.3

* for code like *is>>s*, the string s is seprated by whitespace, and is get the first word;
* for code like getline(is, s), the is get the whole line.
~~~
for example:

cin >> s;
input "  my godness  "
s is "my"

getline(cin, s);
input "  my godness  "
s is "  my godness  "
~~~
the example is exercise3_2.
